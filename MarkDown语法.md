## 文本格式 (Text Formatting)

| **格式**  | **Markdown 语法**         | **效果 (示例)** | **快捷键/说明** |     |
| ------- | ----------------------- | ----------- | ---------- | --- |
| **粗体**  | `**这是粗体**` 或 `__这是粗体__` | **这是粗体**    | 用于强调重要术语。  |     |
| _斜体_    | `*这是斜体*` 或 `_这是斜体_`     | _这是斜体_      | 用于术语或书名。   |     |
| ~~删除线~~ | `~~这是删除线~~`             | ~~这是删除线~~   | 用于标记过时的内容。 |     |
| **粗斜体** | `***粗斜体***`             | **_粗斜体_**   | 强化强调。      |     |
## 标题 (Headings)

使用 `#` 符号来创建不同级别的标题，一个 `#` 是一级标题，最多支持六级。

| **级别**   | **Markdown 语法** | **效果 (示例)**       | **说明** |
| -------- | --------------- | ----------------- | ------ |
| **一级标题** | `# 一级标题`        | **最大号字体，用作文章标题。** |        |
| **二级标题** | `## 二级标题`       | **常用于主要章节。**      |        |
| **三级标题** | `### 三级标题`      | **常用于子部分。**       |        |
| 四级标题     | `#### 四级标题`     |                   |        |
## 列表 (Lists)

### 无序列表 (Unordered List)

使用星号 (`*`)、加号 (`+`) 或减号 (`-`)。

Markdown

```
* 列表项 A
* 列表项 B
  * 嵌套列表项 B-1 (缩进 2 或 4 个空格)
```

### 有序列表 (Ordered List)

使用数字和句点 (`1.`)。

Markdown

```
1. 第一步
2. 第二步
3. 第三步
```
## 代码块 (Code Blocks)

### 行内代码 (Inline Code)

使用单个反引号 (`` ` ``) 包裹短小的代码或命令名称。

Markdown

```
这是一个 `ArrayList` 对象。
```

### 代码块 (Block Code)

使用三个反引号 (` ``` `) 包裹多行代码。在第一个三反引号后添加语言名称，可以实现**语法高亮**（这是 GitHub 笔记中非常重要的一点）。

Markdown

```
​```java
// 这是一个 Java 代码块
public void greet() {
    System.out.println("Hello, World!");
}
​```
```

## 链接与图片 (Links and Images)

### 链接 (Links)

|**格式**|**Markdown 语法**|**效果 (示例)**|
|---|---|---|
|**内联链接**|`[链接文本](URL)`|[Google 搜索引擎](https://www.google.com/)|
|**参考链接**|`[链接文本][ref]`... `[ref]: URL`|[查看 GitHub 首页][github]<br><br>  <br>  <br><br>[github]: https://github.com|

### 图片 (Images)

图片语法与链接相似，但在前面多一个感叹号 (`!`)。

Markdown

```
![图片描述/替代文本](图片 URL 或相对路径)
```

## 引用与分隔 (Quotes and Separators)

### 引用块 (Blockquotes)

使用大于号 (`>`) 标记引用内容。

Markdown

```
> 这是我从书中引用的一个重要概念。
> 它有助于理解 OOP 原则。
```

### 分隔线 (Horizontal Rule)

使用三个或更多星号 (`***`)、减号 (`---`) 或下划线 (`___`)。用于分隔大型章节或主题。

Markdown

```
***
```

## 表格 (Tables)

使用竖线 (`|`) 和减号 (`-`) 来创建表格。第二行用于定义对齐方式。

Markdown

```
| 列标题 1 | 列标题 2 | 列标题 3 |
| :--- | :---: | ---: |
| 左对齐数据 | 居中对齐数据 | 右对齐数据 |
| 更多数据 A | 更多数据 B | 更多数据 C |
```
